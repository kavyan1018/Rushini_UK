Types of inbuilt string functions in python

len => length of string 
upper => convert string to upper case
lower => convert string to lower case
chr => convert ascii to character
ord => convert character to ascii

title => convert first letter of each word to upper case
capitalize => convert first letter of first word to upper case
swapcase => convert upper case to lower case and vice versa
